Aperture is a sleek, responsive image gallery built with **HTML, CSS, and JavaScript**. It features category-based filtering, smooth animations, and a lightbox viewer with keyboard navigation — designed to showcase collections beautifully and professionally.

---

## ✨ Features

- **[Responsive design](ca://s?q=Explain_responsive_design)** — adapts seamlessly across devices  
- **[Category filters](ca://s?q=How_do_category_filters_work)** — toggle between *Nature*, *Urban*, *Wildlife*, or view *All*  
- **[Animated cards](ca://s?q=CSS_card_animation)** — smooth rise-in effect with hover zoom and overlay captions  
- **[Lightbox viewer](ca://s?q=Lightbox_gallery_explained)** — view images full-size with captions  
- **[Keyboard navigation](ca://s?q=Keyboard_navigation_in_web_apps)** — use `←` and `→` arrows to browse, `Esc` to close  
- **[Lazy loading](ca://s?q=Lazy_loading_images)** — optimized performance with `loading="lazy"`
