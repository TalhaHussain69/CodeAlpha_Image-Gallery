# ⚙️ How It Works

- Images are stored inside a JavaScript array.
- JavaScript dynamically creates gallery cards.
- Filter buttons display images by category.
- Clicking an image opens the fullscreen lightbox.
- Navigate using on-screen buttons or keyboard arrow keys.
- Press **Esc** to close the lightbox.

---

| Key | Function |
|------|----------|
